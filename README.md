💱 Exchange Rate Suite — 7 языков, курсы валют онлайн
Коллекция из семи независимых реализаций приложения для отображения курсов валют с богатым функционалом:

💵 Текущие курсы для всех валют или выбранных пар

🔄 Конвертер валют — пересчёт любой суммы

📈 Исторические данные за выбранный период (график)

📊 Визуализация — линейная диаграмма изменения курсов

🌍 Автоопределение валюты по IP (определение страны пользователя)

⭐ Избранные валюты — сохранение в localStorage / файл

🔁 Автообновление каждые N секунд/минут

📅 Поддержка криптовалют (биткоин, эфириум и др.)

📉 Аналитика — изменение за день в процентах

🗺️ Геолокация — определение валюты по местоположению
https://open.er-api.com/v6/latest/{BASE_CURRENCY}

Язык	Интерфейс	Библиотеки / фреймворки	Особенности

Java	Swing GUI	java.net.http, org.json, JFreeChart	Десктопное приложение с графиками, автообновление
C#	WinForms	System.Net.Http, Newtonsoft.Json, LiveCharts	Графики, трей-иконка, экспорт в CSV
